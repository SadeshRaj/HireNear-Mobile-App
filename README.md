# HireNear Mobile Application

## Project Overview

A full-stack mobile application that connects clients with nearby service providers using location-based filtering for job posting and bidding.

## Features

* Job Posting Management
* Bidding & Proposal System
* Contract & Booking Management
* Worker Portfolio Management
* Review & Rating System
* Invoice & Payment Tracking
* Integrated Messaging System

## Tech Stack

* Backend: Node.js + Express.js
* Database: MongoDB
* Frontend: React Native
* Hosting & Deployment: AWS / Render / Railway
